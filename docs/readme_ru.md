# EasyOref

Оповещения о ракетных атаках в Израиле → чат вашей семьи в Telegram.
Настроил один раз — забыл.

[English](../README.md) · [עברית](readme_he.md)

> [!CAUTION]
> EasyOref **не заменяет** официальные оповещения Пикуд а-Орэф, приложение [Цева Адом](https://www.oref.org.il/eng) и [Цофар](https://www.tzofar.com/).
> Бот **дополняет** их — ваши родные за рубежом автоматически узнают, что в вашем городе тревога.
> Всегда следуйте инструкциям Пикуд а-Орэф.

---

## Зачем

Во время ракетной атаки ваши родители видят в новостях «РАКЕТЫ ПО ТЕЛЬ-АВИВУ». Они не знают:

- Это ваш район или 200 км от вас?
- Вы в безопасности?
- Звонить? (вы в мамад, руки заняты)

**Готовых решений нет.** Приложения тревоги — для вас в Израиле.
Ваши родные в Москве / Киеве / Берлине получают автоматическое уведомление в Telegram: город, тип тревоги, что делать — **на русском**.

---

## Как работает

```
Oref API (опрос каждые 2 сек)
  → Фильтр: тревога в МОЁМ городе?
    → Тип: раннее предупреждение / сирена / отбой
      → Кулдаун: не спамить (2 мин / 90 сек / 5 мин)
        → Перевод на ru/en/he/ar
          → Telegram (+ опциональный GIF)
```

Без ИИ, без облаков, без внешних аккаунтов. Задержка — меньше секунды.

---

## Возможности

- **Фильтр по городам** — только ваш город, не вся страна
- **4 языка** — русский, английский, иврит, арабский
- **3 типа тревог** — раннее предупреждение, сирена, отбой
- **GIF-режимы** — `funny_cats` / `assertive` / `none` (котики снижают панику)
- **Кастомные сообщения** — своё описание для каждого типа тревоги
- **Ночной режим** — спокойные GIF с 03:00 до 11:00 (Израиль)
- **Health-эндпоинт** — `/health` для мониторинга

## Быстрый старт

**Нужно:** Docker, токен Telegram-бота, chat ID.

### Шаг 1: Бот в Telegram

1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot` → скопируйте токен
2. Добавьте бота в семейный групповой чат
3. Перешлите любое сообщение из чата в [@userinfobot](https://t.me/userinfobot) → скопируйте chat ID

### Шаг 2: Найдите ID города

Откройте [cities.json](https://raw.githubusercontent.com/eladnava/pikud-haoref-api/master/cities.json) и найдите свой город. Скопируйте число из поля `id`.

Пример: `"id": 722` = Тель-Авив — Юг и Яффо.

### Шаг 3: Запуск

```bash
git clone https://github.com/mikhailkogan17/easyoref.git
cd easyoref
cp config.yaml.example config.yaml
```

Отредактируйте `config.yaml`:

```yaml
# Ваш город (ID из шага 2)
city_ids:
  - 722

# Язык сообщений: ru / en / he / ar
language: ru

# Telegram (из шага 1)
telegram:
  bot_token: "вставьте-токен"
  chat_id: "-1001234567890"
```

Запуск:

```bash
docker compose up -d
```

Проверка:

```bash
curl localhost:3100/health
```

**Готово.** Бот следит за API и отправляет тревоги в ваш чат.

## Конфигурация

Все настройки — в [`config.yaml`](../config.yaml.example).

### Обязательные

| Ключ | Описание |
| --- | --- |
| `city_ids` | Города для мониторинга ([найти ID](https://raw.githubusercontent.com/eladnava/pikud-haoref-api/master/cities.json)) |
| `telegram.bot_token` | Токен от @BotFather |
| `telegram.chat_id` | ID чата (отрицательное число) |

### Опциональные

| Ключ | По умолчанию | Описание |
| --- | --- | --- |
| `language` | `ru` | Язык: `ru` `en` `he` `ar` |
| `alert_types` | все | Типы тревог: `early` `siren` `incident_over` |
| `gif_mode` | `none` | GIF: `funny_cats` `assertive` `none` |
| `title_override.*` | — | Свой заголовок по типу тревоги |
| `description_override.*` | — | Своё описание по типу тревоги |
| `observability.betterstack_token` | — | Логирование через [Better Stack](MONITORING.md) |

## FAQ

**Несколько городов?**
`city_ids: [722, 723, 1]`

**Зачем котики?**
Котик рядом с «оставайтесь у мамада» = «я в порядке». Текст «СИРЕНА В ТЕЛЬ-АВИВЕ» без контекста вызывает панику.

**GIF-ротация сбрасывается при перезапуске?**
Нет — состояние хранится в Docker named volume.
