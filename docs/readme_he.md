# EasyOref

התרעות פיקוד העורף בזמן אמת → צ׳אט המשפחה בטלגרם.
מגדירים פעם אחת, שוכחים.

[English](../README.md) · [Русский](readme_ru.md)

> [!CAUTION]
> EasyOref **אינו מחליף** את התרעות פיקוד העורף, אפליקציית [צבע אדום](https://www.oref.org.il/) או [צופר](https://www.tzofar.com/).
> הבוט **משלים** אותן — מעדכן את המשפחה שלכם בחו״ל שיש אזעקה באזור שלכם.
> תמיד פעלו לפי הנחיות פיקוד העורף.

---

## למה

בזמן מתקפת טילים, ההורים שלכם בחו״ל רואים בחדשות «טילים על תל אביב». הם לא יודעים:

- זה השכונה שלכם או 200 ק״מ משם?
- אתם בטוחים?
- להתקשר? (אתם בממ״ד, ידיים תפוסות)

**אין פתרון קיים.** אפליקציות אזעקה מיועדות לכם בישראל.
ההורים שלכם במוסקבה / קייב / ברלין צריכים משהו אחר — הודעה אוטומטית, מסוננת, בשפה שלהם.

**EasyOref סוגר את הפער:** המשפחה מקבלת עדכונים בזמן אמת, בלי שתעשו כלום.

---

## איך זה עובד

```
Oref API (בדיקה כל 2 שניות)
  → סינון: האזעקה באזור שלי?
    → סיווג: התרעה מוקדמת / צפירה / ירידה ממרחב מוגן
      → קולדאון: לא לספאם (2 דק׳ / 90 שנ׳ / 5 דק׳)
        → תרגום ל-ru/en/he/ar
          → שליחה לטלגרם (+ GIF אופציונלי)
```

בלי AI, בלי ענן, בלי חשבונות חיצוניים. סינון דטרמיניסטי. השהייה של פחות משנייה.

---

## יכולות

- **סינון לפי עיר** — רק האזעקות באזור שלכם
- **4 שפות** — רוסית, אנגלית, עברית, ערבית
- **3 סוגי אזעקה** — התרעה מוקדמת, צפירה, ירידה ממרחב מוגן
- **מצבי GIF** — `funny_cats` / `assertive` / `none` (חתולים מפחיתים פניקה)
- **הודעות מותאמות** — טקסט משלכם לכל סוג אזעקה
- **מצב לילה** — GIF רגוע בין 03:00–11:00
- **Health endpoint** — `‎/health` לניטור

## התחלה מהירה

**צריך:** Docker, טוקן בוט טלגרם, chat ID.

### שלב 1: בוט בטלגרם

1. שלחו הודעה ל-[@BotFather](https://t.me/BotFather) → ‏`/newbot` → העתיקו את הטוקן
2. הוסיפו את הבוט לצ׳אט הקבוצתי
3. העבירו הודעה מהצ׳אט ל-[@userinfobot](https://t.me/userinfobot) → העתיקו את ה-chat ID

### שלב 2: מצאו את ID העיר

פתחו את [cities.json](https://raw.githubusercontent.com/eladnava/pikud-haoref-api/master/cities.json) וחפשו את העיר. העתיקו את המספר מ-`id`.

דוגמה: `"id": 722` = תל אביב — דרום ויפו.

### שלב 3: הפעלה

```bash
git clone https://github.com/mikhailkogan17/easyoref.git
cd easyoref
cp config.yaml.example config.yaml
```

ערכו את `config.yaml`:

```yaml
# העיר שלכם (ID משלב 2)
city_ids:
  - 722

# שפת ההודעות: ru / en / he / ar
language: he

# טלגרם (משלב 1)
telegram:
  bot_token: "הדביקו-טוקן-כאן"
  chat_id: "-1001234567890"
```

הרצה:

```bash
docker compose up -d
```

בדיקה:

```bash
curl localhost:3100/health
```

**זהו.** הבוט עוקב אחרי API של פיקוד העורף ושולח אזעקות לצ׳אט שלכם.

## הגדרות

כל ההגדרות ב-[`config.yaml`](../config.yaml.example).

### חובה

| מפתח | תיאור |
| --- | --- |
| `city_ids` | ערים לניטור ([מצאו ID](https://raw.githubusercontent.com/eladnava/pikud-haoref-api/master/cities.json)) |
| `telegram.bot_token` | טוקן מ-@BotFather |
| `telegram.chat_id` | ID של הצ׳אט (מספר שלילי) |

### אופציונלי

| מפתח | ברירת מחדל | תיאור |
| --- | --- | --- |
| `language` | `ru` | שפה: `ru` `en` `he` `ar` |
| `alert_types` | הכל | סוגי אזעקה: `early` `siren` `incident_over` |
| `gif_mode` | `none` | GIF: `funny_cats` `assertive` `none` |

## שאלות נפוצות

**מספר ערים?**
`city_ids: [722, 723, 1]`

**למה חתולים?**
חתול ליד «הישארו ליד הממ״ד» = «אני בסדר». טקסט «צפירה בתל אביב» בלבד מעורר פניקה.

**רוטציית GIF מתאפסת בהפעלה מחדש?**
לא — השמור ב-Docker named volume.
